Replication Guide

1. Environment Setup

Prerequisites

  • Python 3.8 or higher
  • pip package manager

Installation

  1. Clone this repository to your local machine.
  2. Navigate to the root folder.
  3. Install the required dependencies:
pip install -r requirements.txt

2. Dataset Generation

I generate synthetic data using src/data_gen.py. This script creates random quantum states and simulates projective measurements.

Command:

python src/data_gen.py --train_size 10000 --test_size 1000 --shots 1024

Output:

  • data/train_data.npz
  • data/test_data.npz
  • data/benchmark_data.npz (Standard benchmark states)

3. Model Training

Train the Cholesky-MLP model.

Command:

python src/train.py --epochs 20 --batch_size 32 --lr 0.001

Behavior:

  • Trains for 20 epochs.
  • Validates after each epoch.
  • Saves the model with the lowest validation loss to outputs/best_model.pt.

4. Evaluation and Verification

To audit performance against the assignment requirements (Fidelity and Trace Distance metrics):

Command:

python src/validate_benchmarks.py

Output:

  • Prints table of Fidelity and Trace Distance for |0⟩, |1⟩, |+⟩, |-⟩, |i⟩, |-i⟩.
  • Confirms if mean fidelity meets expectations (>97% with 1024 shots).

Hardware Specs for Recreation

  • I generated the reported results on a standard CPU.
  • Training time: < 2 minutes on CPU.
